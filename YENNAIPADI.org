* How to use
#+begin_src shell
  mkdir ~/code && cd ~/code
  git clone https://github.com/vanangamudi/link-capture.git
#+end_src

Edit this systemd service description to suite your setup, make sure to change the =username=, refer [[*Systemd Service description][here]] to run this service
#+begin_src  conf
  [Unit]
  Description=Common Lisp Link Capture Service
  After=network.target

  [Service]
  Type=simple
  User=<your-username>
  WorkingDirectory=%h/code/link-capture
  ExecStart=ros run -- --script %h/code/link-capture/link-capture
  Restart=always
  RestartSec=3

  [Install]
  WantedBy=multi-user.target
#+end_src

Make the =systemd= aware of this file
#+begin_src shell
  sudo ln -s ~/code/link-capture/link-capture.service /etc/systemd/system/link-capture.service
  sudo systemctl daemon-reload
  sudo systemctl enable link-capture
  sudo systemctl start link-capture
  sudo systemctl status link-capture
#+end_src

Load the browser-extension in your browser, in firefox you can load this by going to =about:debugging= and clicking the =Load Temporary Addon= button. Upload the =manifest.json= under =browser-extension/= directory or you can choose to stick with [[bookmark-let]].
